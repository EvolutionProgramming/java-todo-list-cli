# 📝 Java To-Do List CLI

> Um gerenciador de tarefas via linha de comando desenvolvido para organização pessoal e consolidação de conhecimentos em Java.

---

## 🎯 Sobre o Projeto

A ideia deste repositório nasce tanto da necessidade de **praticar programação**, quanto de uma **necessidade real de organizar minhas tarefas diárias**. 

Este repositório serve como um laboratório prático para o meu desenvolvimento contínuo, onde aplico conceitos do **básico ao avançado** da linguagem. A proposta é escalar o projeto gradualmente, adicionando novas funcionalidades, refatorando o código e aplicando melhores práticas à medida que evoluo meus estudos.

---

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Java 25
* **IDE:** IntelliJ IDEA
* **Interface:** CLI (Interface de Linha de Comando)

---

## 🚀 Funcionalidades Principais *(Em constante evolução)*

* [x] Adicionar novas tarefas
* [ ] Listar tarefas pendentes e concluídas
* [ ] Marcar tarefas como concluídas
* [ ] Remover tarefas

---

## 🚀 Como Executar o Projeto

### Pré-requisitos
* **Java JDK 25** (ou superior) instalado.

### 💻 Rodando na IDE (IntelliJ IDEA)
1. Clone o repositório: `git clone https://github.com/EvolutionProgramming/java-todo-list-cli.git
2. Abra a pasta do projeto no **IntelliJ IDEA**.
3. Abra o arquivo principal contendo o `main` (ex: `Main.java`) e clique no botão **Run** (ícone do play verde ▶️) ou pressione `Shift + F10`.

### 🖥️ Rodando pelo Terminal
```bash
# Compilar os arquivos
javac src/*.java

# Executar
java -cp src Main
